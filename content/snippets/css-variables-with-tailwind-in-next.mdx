---
title: How to use CSS variables with Tailwind CSS in Next 13
publishedAt: 2023-01-02
summary: Setting up a fullstack application with Prisma and a PostgresSQL database in Railway using the Next 13 app directory.
---

1. Add your your variable to the root of the global.css stylesheet

```css
/* global.css */

:root {
  --primary-color: #14342b;
}
```

2. Add the variable to the extended colors property of the theme in the tailwind.confing.js

```js
/* tailwind.config.js */

theme: {
  extend: {
    colors: {
      primary: "var(--primary-color)"
    }
  }
}
```
