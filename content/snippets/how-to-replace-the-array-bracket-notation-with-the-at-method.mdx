## Get last element of an array and last character of a string

```jsx
const teams = ["Giants", "Mets", "Braves", "Astros"]

// bracket notation (array)
const lastElementBracket = teams.[teams.length - 1]

// at method (array)
const lastElementAt = teams.at(-1) // Astros

// at method (string)
const lastCharacterAt = lastElementAt.at(-1) // s
```

## Polyfilling

```jsx
const teams = ["Giants", "Mets", "Braves", "Astros"]

Array.prototype.at = function (index) {
  if (index < 0) {
    return this[this.length - 1]
  }
  return this[index] // this cannot be used with arrow function
}
```
